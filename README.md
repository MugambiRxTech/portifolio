#React Portfolio SPA
#Description

This project is a Single Page Application (SPA) built using React for a summative lab assessment.
The application displays a portfolio of projects for a creative studio or developer.

#Features

View a list of projects on the home page

Add new projects using a form

Search and filter projects by category

Delete projects

View project details using client-side routing

Data is saved using local storage

Responsive design for different screen sizes

#Technologies Used

React (Vite)

React Router DOM

JavaScript

CSS

#How to Run the Project

Install dependencies

npm install


Start the development server

npm run dev


Open in the browser

http://localhost:5173
