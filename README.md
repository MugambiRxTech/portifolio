# React Portfolio SPA

## Description
This project is a Single Page Application (SPA) built using React as part of a summative lab assessment.
The application is a portfolio platform that displays projects for a creative studio or developer.

## Features
- Display a list of projects on the home page
- Add new projects using a form
- Search and filter projects by category
- Delete projects
- View project details using client-side routing
- Save project data using local storage
- Responsive design for mobile and desktop

## Technologies Used
- React (Vite)
- React Router DOM
- JavaScript
- CSS
